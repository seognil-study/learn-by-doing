# Learn Cypress

## Start From Scratch

```bash
mkdir cy-test
cd cy-test

```

```bash
# by npm
npm init -y
npm i cypress --save-dev
npx cypress open

# or by yarn
yarn init -y
yarn add -D cypress
npx cypress open
```

Check [Opening Cypress](https://docs.cypress.io/guides/getting-started/installing-cypress.html#Opening-Cypress)
